### Precious Report Overview

## Objective
The objective of this assignment was to build a story on the user of the Precious.E01 image. It was also done to help develop my overall digital forensics skills. 

## Skills Gained
- An understanding of how disk images are created and validated using hash values.
- Interpreting NTFS (Windows) file systems.
- Identifying and analyzing key artifacts.
- Creating and keeping a chronological timeline of evidence.
- Using digital forensic tools such as Autopsy, FTK Imager, Registry Viewer, Registry Explorer, and Windows Event Viewer.

## Tools Used

| Tool                   | Version        | Used For                                        |
|------------------------|----------------|-------------------------------------------------|
| Autopsy                | 4.22.1         | A digital forensics platform used to analyze hard drives or images of hard drives.|
| Registry Explorer      | 2.1.0          | An enhanced Windows registry analysis tool used to examine and interpret registry hives.|
| Registry Viewer        | Demo Mode      | A software tool used to access, read, and analyze the Windows Registry. |
| Windows Event Viewer   | Windows 11     | A Windows software used to view detailed logs of events on a Windows system. |
| PicPick                | 7.4.0          | A software to edit photos, so it is easier to see important information. |

## Required Flags 

1)	User Account Discovery: What is the name of the main Windows user account?
2)	Operating System Version: What operating system is installed?
3)	Time Zone: What time zone is the computer set to?
4)	USB Device History: Find one USB storage device that was connected to the computer and provide the make/model/serial number.
5)	Browser History: Identify one website the user visited that could indicate personal interests.
6)	Email Attachments: Find at least one email attachment that the user sent.
7)	Recent Documents: Find one document that the user recently opened.
8)	Email Artifacts: Locate at least one email address used on the computer.
9)	Pictures: Identify at least three images that could be meaningful.
10)	Software: Identify at least three programs that are installed on the computer.
11)	System Boot: When is the last recorded system shutdown date and time?
12)	Deleted Files: Identify and recover a deleted file.
13)	Did you discover any other interesting evidence on the computer?

## Bonus Flags

14)	Find evidence that the computer printed a document. What was the document? What do you know about it? (This is a bit tricky and will require outside research.)
15)	Does this user seem to like music? What evidence supports your answer?
